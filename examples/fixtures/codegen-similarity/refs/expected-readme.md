# Fibonacci

A Python module that calculates Fibonacci numbers.

## Usage

```python
from fib import fibonacci

# Get the 10th Fibonacci number
print(fibonacci(10))  # 55
```

## Functions

### `fibonacci(n)`

Returns the nth Fibonacci number. Accepts a non-negative integer and returns an integer. Raises `ValueError` if n is negative.

## Examples

```python
fibonacci(0)   # 0
fibonacci(1)   # 1
fibonacci(10)  # 55
fibonacci(20)  # 6765
```
